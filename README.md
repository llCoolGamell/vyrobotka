# Выработка по сотрудникам

Программа с современным интерфейсом: загружает Excel-файл **«сводный запрос»** и
формирует на каждого сотрудника отдельный Excel-файл с его работами и количеством.

## Что делает

1. Читает исходный файл со столбцами
   `Исполнитель | Роль | Киз | Boxcount_короба | Processcount_строки | Строк`.
2. Объединяет строки с одинаковыми **ФИО + Роль** и складывает числовые значения.
3. Считает столбец **«объём выполненных работ»** = сумма всех чисел строки.
4. Сортирует по ФИО и роли.
5. Сохраняет на каждого сотрудника отдельный `.xlsx` (имя файла = ФИО),
   с итоговой строкой «ИТОГО» и оформлением в зелёном стиле.

Колонки итоговых файлов:
`Исполнитель | Роль | Киз | короба | сборка_строк | контроль_строк | объём выполненных работ`

## Интерфейс

* **Слева** — зона загрузки: перетащите файл мышью или нажмите «Загрузить файл…».
* **Справа** — выбор папки для выгрузки и большая зелёная кнопка
  **«СОЗДАТЬ СВОДНЫЙ ЗАПРОС»** с индикатором прогресса.

## Установка и запуск

```bash
pip install -r requirements.txt
python app.py
```

> Перетаскивание мышью работает при установленном `tkinterdnd2`.
> Если его нет — файл всё равно можно выбрать кнопкой.

## Сборка в .exe (Windows)

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed ^
  --collect-all customtkinter ^
  --collect-all tkinterdnd2 ^
  --name "Выработка" app.py
```

Готовый `Выработка.exe` появится в папке `dist/`.
Запускается двойным кликом, Python на компьютере не нужен.

## Структура

```
app.py            — интерфейс (customtkinter)
processor.py      — логика обработки Excel
requirements.txt  — зависимости
example/          — пример входного и выходного файлов
```
